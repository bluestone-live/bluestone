# Mobile-FE-3

## requirements

- nodejs >= 8

## Install Dependencies

```shell
yarn
```

## Environment Configuration

rename .env.example to .env.dev(for development) or .env.prod(for production)

## Run Dev

```shell
yarn dev
```

## Scripts

- Update translations

```
yarn translations
```
